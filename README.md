# Waves2 - Локальная копия проекта Balansity

Это локальная копия проекта Balansity с эмуляцией базы данных. Проект включает:
- Полный лендинг
- Процесс регистрации
- Онбординг
- Попадание в личный кабинет

## Особенности

- **Локальная база данных**: Использует IndexedDB для хранения данных локально
- **Эмуляция Supabase**: Полная эмуляция API Supabase без необходимости облачного сервера
- **Сохранение данных**: Все данные сохраняются локально в браузере

## Установка

1. Установите зависимости:
```bash
npm install
```

2. Запустите dev сервер:
```bash
npm run dev
```

3. Откройте браузер на `http://localhost:8080`

## Использование

### Регистрация

1. Перейдите на страницу регистрации
2. Заполните форму регистрации
3. После регистрации вы будете перенаправлены на онбординг

### Онбординг

Процесс онбординга включает:
- Заполнение профиля
- Выбор региона
- Настройка семьи
- Указание беспокойств
- Прохождение чек-апа

### Личный кабинет

После завершения онбординга вы попадете в личный кабинет.

## База данных

База данных использует IndexedDB и сохраняется локально в браузере. Все данные доступны только в текущем браузере.

### Структура данных

- **users**: Пользователи
- **profiles**: Профили членов семьи
- **assessments**: Оценки и чек-апы
- **answers**: Ответы на вопросы
- **appointments**: Записи на консультации
- **payments**: Платежи

## Технологии

- React 18
- TypeScript
- Vite
- React Router
- TanStack Query
- Tailwind CSS
- shadcn/ui
- IndexedDB (idb)

## Примечания

- Проект работает полностью локально без необходимости облачного сервера
- Данные сохраняются в IndexedDB браузера
- Для использования реального Supabase установите переменные окружения `VITE_SUPABASE_URL` и `VITE_SUPABASE_ANON_KEY`
